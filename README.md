# Мой интернет-магазин на Flask

Это учебный проект простого онлайн-магазина, созданный на Python с использованием фреймворка Flask.

## Как запустить проект локально

1. Скачайте файлы или клонируйте репозиторий.
2. Установите необходимые библиотеки (Flask и Gunicorn), используя ваш файл зависимостей:
   ```bash
   pip install -r requirements.txt
